# TPmath

A practical Python application engineered to help consumers navigate complex retail math.

### Purpose
TPmath lets the user demystify corporate pricing schemes to help find the best price on Toilet Paper. By breaking down "mega roll" configurations and sheet counts into a standard metric, it ensures you know exactly what you are paying per unit.

### Features
* **Unit Cost Breakdown:** Input package counts, sheet sizes, or roll counts to uncover the true value.
* **Corporate Pricing Defense:** Strips away confusing marketing labels to reveal the actual cost comparison.
* **Mobile Ready:** Configured via Buildozer for deployment to mobile devices for use directly in the store aisle.
